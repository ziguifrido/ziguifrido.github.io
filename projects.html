Ajuda-me

(Em Desenvolvimento)

Ajude-me é um projeto que visa facilitar o contato entre doadores, voluntários e ONGs. Para tanto ele disponibiliza dados das ONGs como trabalhos realizados, tipos de favorecidos, tipos de necessidades, informações de contato e localização. Através dele o cidadão pode facilmente localizar ONGs em sua região e dessa forma incentivar e facilitar a ajuda ao próximo. Para acessar clique aqui. O código fonte pode ser acessado no GitHub.

GitHub Search Web Extension

GitHub Search Web Extension é uma extensão de navegador criada para facilitar a busca por perfis e repositórios no GitHub. Ao clicar no ícone da extensão é aberto um popup com um campo de pesquisa e três opções: abrir perfil, mostrar a imagem do perfil, e pesquisar. A extensão está disponível para Chrome e Firefox. O código fonte pode ser acessado no GitHub.

QRCode Web Extension

QRCode Web Extension é uma extensão de navegador criada para facilitar o compartilhamento de uma URL de um navegador desktop para um smartphone. Ao clicar no ícone da extensão é aberto um popup com um QRCode com a url da aba aberta no navegador. Ao ler o QRCode por um aplicativo leitor de QRCode no smartphone a mesma URL aberta no computador será aberta no smartphone. A extensão está disponível para Chrome e Firefox. O código fonte pode ser acessado no GitHub.
